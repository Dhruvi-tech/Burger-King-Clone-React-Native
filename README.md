## 🚀 Features  

✨ Category-Based Menu Browsing  
🛒 Add to Cart with Smart Quantity Control  
➕➖ Increase / Decrease Quantity  
💰 Dynamic Price Calculation  
📦 Smooth Checkout System  
📜 Order History  
🗑️ Swipe to Remove Items  
⚡ Fast & Responsive UI  

---

## 🧭 App Flow  

```
Home → Menu → Cart → Checkout → Orders
```
