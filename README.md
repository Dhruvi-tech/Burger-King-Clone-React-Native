## рҹҡҖ Getting Started

<br/>

### Prerequisites

```
вң…  Node.js LTS
вң…  React Native CLI environment  вҶ’  https://reactnative.dev/docs/environment-setup
вң…  Android Studio  (for Android)
вң…  Xcode           (for iOS)
```

<br/>

### Installation

```bash
# рҹ“Ҙ Clone the repo
git clone https://github.com/your-username/burger-king-clone.git
cd burger-king-clone

# рҹ“Ұ Install dependencies
npm install

# рҹҚҺ iOS only вҖ” install CocoaPods
cd ios && pod install && cd ..
```

<br/>

### Run the App

```bash
# вҡЎ Start Metro bundler
npx react-native start

# рҹӨ– Android
npx react-native run-android

# рҹҚҺ iOS
npx react-native run-ios
```
